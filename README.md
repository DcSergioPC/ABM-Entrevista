## Pasos para correr el proyecto
Abrir la terminal o consola y pegar lo siguiente segun el sistema operativo
### Linux
```
git clone -b out-of-time https://github.com/DcSergioPC/ABM-Entrevista &&
cd ABM-Entrevista &&
npm install &&
npm run dev
```
### Windows
```
git clone -b out-of-time https://github.com/DcSergioPC/ABM-Entrevista ;
cd ABM-Entrevista ;
npm install ;
npm run dev
```
